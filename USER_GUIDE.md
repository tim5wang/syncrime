# SyncRime v1.0 - 用户指南

## 📱 应用介绍

SyncRime 是一款智能输入助手，集成了 AI 功能帮助你更高效地输入和管理知识。

## ✨ 核心功能

### 1️⃣ 首页 - 输入统计
- 📊 实时查看今日输入记录
- 📈 总输入统计
- ℹ️ 无障碍服务状态监控

### 2️⃣ 搜索 - 快速查找
- 🔍 搜索输入记录
- 📚 搜索知识剪藏
- 🏷️ 按分类筛选结果

### 3️⃣ 知识库 - 知识管理
- 📖 查看所有剪藏内容
- 🏷️ 分类筛选
- 📊 统计卡片（总数/今日新增）

### 4️⃣ AI 助手 - 智能创作 🤖
- ✍️ **智能续写** - 根据上下文自动续写
- 📝 **智能摘要** - 快速总结长文本
- 💬 **AI 对话** - 与 AI 助手聊天
- 🎨 **文本润色** - 多风格优化（专业/正式/轻松）

### 5️⃣ 设置 - 个性化配置
- ⚙️ 无障碍服务设置
- 📊 数据采集配置
- 💾 数据存储管理
- ℹ️ 关于信息

## 🚀 快速开始

### 第一步：安装应用
1. 下载 `SyncRime-v1.0-debug.apk`
2. 允许"未知来源应用"
3. 点击安装

### 第二步：开启无障碍服务
1. 打开 SyncRime 应用
2. 进入"设置"标签页
3. 点击"打开无障碍设置"
4. 找到"SyncRime Input Capture Service"
5. 开启服务

### 第三步：配置 AI 功能（可选）
1. 进入"AI"标签页
2. 点击"配置 API Key"
3. 输入阿里云百炼 API Key
4. 保存后即可使用 AI 功能

**获取 API Key：**
- 访问：https://bailian.console.aliyun.com/
- 创建/选择应用
- 复制 API Key

## 💡 使用技巧

### AI 续写
1. 在"AI"标签页选择"续写"模式
2. 输入上下文（可选）
3. 输入你想续写的内容开头
4. 点击"智能续写"
5. AI 会自动完成创作

### AI 摘要
1. 选择"摘要"模式
2. 粘贴长文本
3. 设置摘要长度（默认 100 字）
4. 点击"智能摘要"
5. 快速获取核心内容

### AI 对话
1. 选择"对话"模式
2. 在输入框输入问题
3. 点击发送或按回车
4. AI 会实时回复

### 文本润色
1. 选择"润色"模式
2. 输入需要优化的文本
3. 选择风格（专业/正式/轻松）
4. 点击"文本润色"
5. 获得优化后的版本

## ⚙️ 设置说明

### 数据采集
- **自动保存**：开启后自动保存所有输入
- **隐私过滤**：自动过滤密码、验证码等敏感信息

### 数据存储
- **清理旧数据**：删除 30 天前的记录
- **导出数据**：备份所有数据到本地

## ❓ 常见问题

### Q: 无障碍服务无法开启？
A: 请确保在手机设置中授予了 SyncRime 无障碍权限。

### Q: AI 功能无法使用？
A: 需要配置阿里云百炼 API Key，请参考"配置 AI 功能"步骤。

### Q: 数据会上传到云端吗？
A: 不会。所有数据都存储在本地，保护你的隐私。

### Q: 如何备份数据？
A: 在"设置"->"数据存储"中点击"导出数据"。
