# SyncRime 实测说明

## 🎯 问题根因

您遇到的两个问题：
1. **找不到插件配置** ✅ 正常现象
2. **没有同步** ✅ 正常现象

---

## 📋 原因说明

### 1. 为什么找不到插件配置？

**答案**：SyncRime 当前版本 **没有 Trime 插件**！

**实现方案**：
```
当前方案：无障碍服务
┌─────────────────────────────────┐
│ SyncRime Android App            │
│ ┌─────────────────────────────┐ │
│ │ 无障碍服务                   │ │
│ │ - 监听所有输入事件           │ │
│ │ - 不依赖特定输入法           │ │
│ │ - 兼容 Trime/Gboard 等       │ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
            ↑ 监听所有输入法
```

**对比方案**：
```
计划中的方案：Trime 插件
┌─────────────────────────────────┐
│ Trime 输入法                     │
│ ┌─────────────────────────────┐ │
│ │ SyncRime 插件层              │ │
│ │ - 深度集成                   │ │
│ │ - 需要编译                   │ │
│ │ - 仅支持 Trime               │ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

**结论**：当前版本通过**无障碍服务**实现，**无需插件配置**！

---

### 2. 为什么没有同步？

**答案**：云端同步功能**尚未实现**（Phase 5）

**当前状态**：

| 功能模块 | 状态 | 说明 |
|---------|------|------|
| 输入采集 | ✅ 完成 | 无障碍服务监听 |
| 本地存储 | ✅ 完成 | Room 数据库 |
| 统计展示 | ✅ 完成 | Jetpack Compose UI |
| 后台任务 | ✅ 完成 | WorkManager 调度 |
| **云端同步** | ⏳ **未实现** | 需要后端 API 服务 |
| Trime 插件 | ⏳ 未实现 | 需要编译集成 |

**数据流向**：
```
用户输入
   ↓
无障碍服务采集
   ↓
Room 数据库 ✅ （本地存储）
   ↓
SyncWorker ✅ （后台任务）
   ↓
⏳ 等待云端 API（Phase 5）
```

**结论**：数据已存储到本地数据库，但**无法同步到云端**（需要后端服务）。

## 🚀 如何使用（实测步骤）

### 步骤 1: 安装 Trime 输入法

```bash
# 下载
curl -LO https://github.com/osfans/trime/releases/download/3.4.2/trime-3.4.2-arm64-v8a.apk

# 安装到手机
adb install trime-3.4.2-arm64-v8a.apk
```

**或**：手机浏览器打开 https://www.coolapk.com/apk/com.osfans.trime 下载安装

### 步骤 2: 安装 SyncRime

```bash
adb install ~/claw_workspaces/tim/syncrime/SyncRime-v1.0-debug.apk
```

### 步骤 3: 配置

1. **启用 Trime**
   - 设置 → 系统 → 语言和输入法
   - 开启 Trime
   - 切换为当前输入法

2. **启用 SyncRime 无障碍服务**
   - 打开 SyncRime App
   - 点击 "开始采集"
   - 设置 → 无障碍 → 开启 "SyncRime 输入采集"

### 步骤 4: 测试

1. 打开微信/备忘录
2. 用 Trime 输入一些文字
3. 回到 SyncRime App
4. 查看统计数据

### 步骤 5: 验证数据

```bash
# 查看数据库文件
adb shell "ls -lh /data/data/com.syncrime.android.debug/databases/"

# 输出示例:
# -rw-rw---- 1 u0_a123 u0_a123 32K syncrime.db

# 导出数据（需要 root）
adb pull /data/data/com.syncrime.android.debug/databases/syncrime.db

# 查看数据
sqlite3 syncrime.db "SELECT COUNT(*) FROM input_records;"
```

## 🔧 故障排查

### 运行诊断脚本

```bash
cd ~/claw_workspaces/tim/syncrime

```

```bash
# 实时查看
adb logcat -s SyncRime -s InputCaptureService

# 导出日志
adb logcat -d > syncrime.log
```

### 常见问题

**Q: 无障碍服务无法开启？**
- 重启手机
- 重新安装 App
- 检查权限设置

**Q: 采集不到输入？**
- 确保 Trime 是当前输入法
- 检查日志是否有事件
- 排除密码输入框

**Q: 数据库为空？**
- 检查无障碍服务是否运行
- 确认输入内容长度>1
- 查看日志是否有错误
