## 产品简介

**SyncRime** 是一款智能输入内容采集与同步系统，与 Trime 输入法深度集成，提供智能化的输入体验。

### 核心特性

- **智能输入采集**: 自动采集输入内容用于智能推荐
- **云端同步**: 安全同步输入数据到云端
- **智能推荐**: 基于上下文的个性化输入建议
- **多语言支持**: 支持 11 种语言的智能切换
- **隐私保护**: 本地数据加密，保护用户隐私

## 快速开始

### 安装步骤

1. **安装 Trime 输入法**
   - 从应用商店下载并安装 Trime (同文输入法)
   
2. **安装 SyncRime 插件**
   - 下载 SyncRime 插件 APK
   - 安装到设备

3. **启用插件**
   - 打开 Trime 设置
   - 找到 SyncRime 插件
   - 启用插件

4. **基本配置**
   - 打开 SyncRime 应用
   - 完成初始设置
   - 授予必要权限

## 功能介绍

### 1. 输入采集

SyncRime 自动采集您的输入内容，用于：
- 智能输入推荐
- 个性化学习
- 输入习惯分析

**状态**: 可在设置中随时开启/关闭

### 2. 云端同步

自动同步功能：
- 增量同步：每次输入后自动同步
- Wi-Fi 仅同步：节省流量
- 手动同步：按需同步

### 3. 智能推荐

基于以下维度的智能推荐：
- 输入历史
- 当前位置
- 当前应用
- 时间上下文
- 用户习惯

### 4. 统计信息

查看您的输入统计：
- 输入总数
- 同步次数
- 使用时长
- 输入习惯

## 设置指南

### 采集设置

| 设置项 | 说明 |
|--------|------|
| 启用输入采集 | 开启/关闭输入采集功能 |
| 采集模式 | 选择采集模式 |

### 同步设置

| 设置项 | 说明 |
|--------|------|
| 自动同步 | 开启后自动同步数据 |
| 同步间隔 | 设置自动同步间隔 (默认 15 分钟) |
| Wi-Fi 仅同步 | 仅在 Wi-Fi 下同步 |

### 通知设置

| 设置项 | 说明 |
|--------|------|
| 启用通知 | 接收同步状态通知 |
| 同步通知 | 同步完成时通知 |
| 推荐通知 | 输入推荐提醒 |

## 权限管理

SyncRime 需要以下权限：

### 通知权限
- **用途**: 接收同步状态和应用更新通知
- **必需**: 是

### 存储权限
- **用途**: 备份和恢复输入数据
- **必需**: 可选

### 位置权限
- **用途**: 基于位置的智能输入建议
- **必需**: 可选

### 管理权限

1. 打开 SyncRime 应用
2. 进入 **设置**
3. 点击 **权限管理**
4. 查看和管理各项权限

## 隐私与安全

### 数据加密

- 本地数据使用 AES-256 加密
- 传输数据使用 TLS 加密
- 云端数据加密存储

### 隐私模式

启用隐私模式后：
- 隐藏敏感输入内容
- 不采集特定应用的数据
- 增强隐私保护

### 数据控制

- 用户拥有所有数据
- 可随时删除本地数据
- 可导出/导入数据

## 常见问题

### Q1: 为什么需要通知权限？

通知权限用于：
- 同步状态提醒
- 输入推荐通知
- 应用更新提示

### Q2: 我的数据安全吗？

是的，我们采用：
- 本地加密存储
- 传输加密
- 隐私模式
- 用户数据控制

### Q3: 如何禁用输入采集？

1. 打开 SyncRime 应用
2. 进入 **设置**
3. 关闭 **启用输入采集**

### Q4: 如何同步数据？

- **自动同步**: 开启自动同步后自动同步
- **手动同步**: 在主界面点击同步按钮

### Q5: 支持哪些语言？

支持 11 种语言：
- 中文 (简体/繁体)
- English
- 日本語
- 한국어
- Français
- Deutsch
- Español
- Русский
- العربية
- Português
- Italiano
